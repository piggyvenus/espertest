HQFileSystemService

The following resources are applicable:

  • /vfs/file
  • /vfs/file/{path}
  • /vfs/multi

/vfs/file

GET

Read top directory.

Response Body

element: fileDetail
media types: application/json

file detail for top directory

/vfs/file/{path}

The following operations are supported on this resource:

  • GET
  • DELETE
  • PUT
  • POST

GET

Read file detail of any single file or directory underneath the top directory.

The REST subpath defines the file path and file name. For example, path '/file/mydir/myfile.txt" means read the file "myfile.txt" residing in directory "mydir".

Parameters

name description type default
path path of the file to read path

Response Body

element: fileDetail
media types: application/json

file detail for the file

DELETE

Delete a file.

Parameters

name description type default
path path of the file (or directory) to delete path

PUT

Update a file.

Parameters

name description type default
path path of the file to update path

Request Body

element: fileDetail
media types: application/json

file body

Response Body

element: fileAbstract
media types: application/json

file abstract of the updated file

POST

Create a file or directory.

Parameters

name description type default
path path of the file or directory to create path

Request Body

element: fileDetail
media types: application/json

contents of the file, or flag indicating that the new file is a directory

Response Body

element: fileAbstract
media types: application/json

file abstract
