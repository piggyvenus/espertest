POST

Deploy a new deployment.

An existing deployment by the same module name or deployment id can be automatically undeployed before the new deployment gets deployed, by passing the undeploy-existing flag.

Parameters

name description type default
endpointName endpoint name path
engineURI engine uri to deploy to path
undeployexisting indicator whether to undeploy an existing deployment query

Request Body

element: deploymentDetail
media types: application/json

dpeloyment detail

Response Body

element: deploymentDetail
media types: application/json

deployment detail of the deployment
