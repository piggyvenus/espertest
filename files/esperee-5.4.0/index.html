Esper Enterprise Edition: Enterprise ready Event Processing and CEP platform

Esper provides Complex Event Processing and Event Series Analysis, commercially supported by EsperTech. Esper Enterprise Edition adds the following additional features:

  • A multi-window, rich, browser-based client application utilizing HTML 5 and JavaScript.
  • A scalable, robust push service for subscription management and data push.
  • A set of composable continuously-updating displays termed eventlets.
  • jQuery Plugin for eventlet activation and extension API for custom eventlet JavaScript.
  • A lightweight server that supports hot deployment of EPL modules and packaged CEP applications.
  • Web application archives for deployment to web tier and server tier web application servers of your choice.
  • Server-to-server socket transport.
  • Distributed cache technology integration.
  • A JMX server endpoint for runtime management.
  • A JDBC driver and server endpoint for interoperability with reporting and query tools.
  • Additional examples.


Running - Getting Started

Please follow the instructions below. See additional detailed instructions and troubleshooting if having issues.

The steps below require Java 2 Standard Edition Runtime Environment (JRE) version 7.0 (JDK 7) or later (download here). Make sure the JRE_HOME or JAVA_HOME environment variable is set.

Navigate to the "bin" folder of the distribution and execute "startup.bat" (Windows) or "startup.sh" (Unix).

After startup, please navigate to:

  • Launch EsperHQ rich client application

The Getting started section is a good place to start. Also consider skimming through the overview chapters of each manual.
