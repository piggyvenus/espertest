Sample Online Shop Dashboard (IFRAME Version)

The displays herein are continuously-updating, as soon as the example data simulator generates data, and following quality of service levels as configured by push services.

Please consult the server documentation examples chapter linked from the index page for more information. This page utilizes HTML IFrames to load individual eventlets scattered over a HTML page. Please also consider using the jQuery eventlet plugin API for integrating eventlets in HTML pages.
