Sample Online Shop Dashboard - Dashboard Page Builder Example

The displays herein are continuously-updating, as soon as the example data simulator generates data, and following quality of service levels as configured by push services.

This page was built using the Dashboard Page Builder feature in the EsperHQ client. We added additional eventlet height parameters and CSS style information to the generated HTML page. Please consult the server documentation examples chapter linked from the index.html page in the root installation folder for more information.

Displays average stock quantity per category as an in-place grid (see Example 2 : Inplace grid of Statement : OnlineShopExample-AvgStockQtyPerCategory)
Displays average stock quantity per category as a column chart grid (see Example 1 : Column Chart : OnlineShopExample-AvgStockQtyPerCategory)
Displays average stock quantity overall as a gauge (see Example 5 : Gauge : OnlineShopExample-AvgStockQtyOverall)
Displays all browse events with low stock quantity in an append-grid (see Example 3 : Appended grid of Statement : OnlineShopExample-AllBrowseEventsLowStock)
Displays all browse events with low stock quantity in a timeline (see Example 4 : Timeline : OnlineShopExample-AllBrowseEventsLowStock)
