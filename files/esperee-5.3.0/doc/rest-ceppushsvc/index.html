Introduction

Usage guidelines:

General usage information can be found in the Enterprise Edition server documentation manual under REST services overview.

The REST context for the services described herein is: /ceppushapi/v1

Services overview:

URI Operation Description
/session POST Create session
/session/{sessionId} PUT Renew/update session
/session/{sessionId} DELETE Remove session
/subscription POST Create subscription
/subscription/{subscriptionId}/control POST Control subscription: Start (if stopped) or refresh (if started) subscription
/subscription/{subscriptionId} DELETE Remove subscription
/pushinfo GET Audit/Monitoring information

REST Resources

This API supports a Representational State Transfer (REST) model for accessing a set of resources through a fixed set of operations. The following resources are accessible through the RESTful model:

  • PushInfoService
  • SessionService
  • SubscriptionService
