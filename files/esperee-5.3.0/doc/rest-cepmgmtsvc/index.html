Services overview:

URI Operation Description
/engine GET Returns engine instances
/engine/{engine} GET Returns engine information
/engine/{engine}/memory GET Returns engine statement-memory-use breakdown
/engine/{engine}/statement GET Returns all statements, add "?filter" for filter criteria expression, , add "?soda=true" to obtain model objects
/engine/{engine}/statement POST Create statement
/engine/{engine}/statement/{statementName} GET Returns statement detail
/engine/{engine}/statement/{statementName} PUT Transition statement
/engine/{engine}/statement/{statementName} DELETE Transition statement to destroyed
/engine/{engine}/statement/{statementName}/iterate GET Returns statement iterate rows
/engine/{engine}/statementvalidation POST Validate an EPL statement
/engine/{engine}/deployment GET Returns deployments
/engine/{engine}/deployment POST Create deployment
/engine/{engine}/deployment/{deploymentId} GET Returns deployment detail
/engine/{engine}/deployment/{deploymentId} PUT Update deployment state
/engine/{engine}/deployment/{deploymentId} DELETE Undeploy and remove deployment
/engine/{engine}/deploymentvalidation POST Validate deployment
/engine/{engine}/eventtype GET Returns list of event types, use "names=true" to get only names
/engine/{engine}/eventtype/{eventTypeName} GET Returns event type
/engine/{engine}/namedwindow GET Returns named window info
/engine/{engine}/ondemandquery POST Perform on-demand query
/engine/{engine}/variable GET Get variable values
/engine/{engine}/variable/{variableName} PUT Update variable value
/engine/{engine}/dataflow GET Get dataflows
/engine/{engine}/dataflow/{dataflowName} GET Get dataflow
/engine/{engine}/dataflowinstance GET Get dataflow instances
/engine/{engine}/dataflowinstance POST Instantiate dataflow
/engine/{engine}/dataflowinstance/{dfInstName} GET Return dataflow instance detail
/engine/{engine}/dataflowinstance/{dfInstName} PUT Cancel/Update dataflow instance
/engine/{engine}/dataflowinstance/{dfInstName} DELETE Cancel and remove dataflow instance
/engine/{engine}/scenariorun POST Execute a scenario of EPL and scenario
/engine/{engine}/sequencevalidation POST Validate a sequence
/engine/{engine}/sequenceexecution GET Returns sequence executions
