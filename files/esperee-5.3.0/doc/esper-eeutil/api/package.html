Enterprise Edition Domain Model Classes.

Domain model classes are REST data model classes as well as Push Services messages. The root package is com.espertech.esper.eeutil.domain.

Classes herein may change between versions however we are not expecting substantial change.

Shared or common data model classes are in package com.espertech.esper.eeutil.domain.common.

REST web services return bodies with error information encoded in JSON, the data model classes for these are found in com.espertech.esper.eeutil.domain.service.

CEP Management data model classes are found in com.espertech.esper.eeutil.domain.mgmt.

CEP Push Services data model classes are found in com.espertech.esper.eeutil.domain.push.

CEP Push Services message classes are found in com.espertech.esper.eeutil.domain.message.
