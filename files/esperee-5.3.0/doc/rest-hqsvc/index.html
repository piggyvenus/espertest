Introduction

Usage guidelines:

General usage information can be found in the Enterprise Edition server documentation manual under REST services overview.

The REST context for the services described herein is: /hqapi/v1

Services overview:

URI Operation Description

/endpoint/{endpoint}/engine/{engine}/deployment/{deploymentId} PUT Update an endpoint's deployment state

/endpoint/{endpoint}/engine/{engine}/sequenceexecution POST Create a endpoint's executing sequences

/endpoint/{endpoint}/engine/{engine}/sequenceexecution/{sequenceName} PUT Stop and start endpoint's executing sequences
/endpoint/{endpoint}/engine/{engine}/sequenceexecution/{sequenceName} DELETE Stop and remove endpoint's executing sequences
/eventlet/parsexml POST Post eventlet xml and receive a JSON document back
/eventlet/parsejson POST Post eventlet json and receive an XML document back

/moduletemplate GET Returns module templates
/resource POST Accepts a list of resource addresses, returns a list of resource details
/vfs/file/.* GET Directory listing or file body at server
/vfs/file/.* POST Create a directory at new folder path, or create a file
/vfs/file/.* PUT Replace contents of text file at path
/vfs/file/.* DELETE Delete file or folder
/vfs/multi GET Get multiple files in one call, url parameter "file" for each file
